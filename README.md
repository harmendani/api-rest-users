# api-rest-users
API RESTful for signin/signup users using Express, JWT and MongoDB.

# Run app

- Create a .env file on root:
PORT=XXXX .. (Number)
JWT_SECRET=XXXXXXXX.. (HEX String 32 Bytes)
DATABASE_NAME=myapp (Name of BD)
- npm install
- pm2 start

# Logs
 - pm2 logs
